Guide: First Time Roach Setup
Step 1: Welcoming Roach

Add Roach To Your Discord

Step 2: Adjusting Roles

In order for Roach to properly function, you need to ensure that the Roach role is set above all roles, like this: Do know that Roach does not need to be at the very top, just simply above the roles you intend to have as role reactions.

Step 3: Add a Text Channel for Roach

Next, you need to create a channel for Roach. A common example would be to create a Text Channel called #roles. You can always change it later here if you create a new channel.
Note: When you finish your changes, you should immediately notice the change (either the message was edited or a new one was created) in the channel.

Step 4: Add Roles and Emotes

Lastly, add any roles and emotes you would like to associate/combine. You can always come back to this website and update the roles or emotes respectively. Remember to submit all changes before testing if it works!

Step 5: Use Our Website

The goal of this website was to remove any obnoxious commands you would have to make in Discord itself. You should be able to update Roach, if setup properly, on this website with just your Discord account.
